## ❄️ Shirokuma studioへようこそ！

私たちは、技術への情熱を持った学生たちが主体となって運営しているGitHub Organizationです。
スマートでUI/UXに配慮した次世代のWebサービス、Minecraft Mod、Discord Bot、そして様々なアプリケーションを、
オープンソースで開発しています。

新しい技術を学び、仲間と協力しながら、世界を少しだけ面白く、そして便利にすることを目指しています。
